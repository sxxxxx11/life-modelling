# Life Modelling

> Rewrite the underlying code of your life

**Life Modelling** is a free, statistics-based life trajectory simulator. It doesn't sell anxiety or sugarcoat the future — by analyzing your current daily habits, it uses probabilistic models to project your future life trajectory and identify the single parameter most worth changing.

[→ Try Now](https://sxxxxx11.github.io/life-modelling/)

---

## ✨ Core Features

### 1. See Where You Stand
Enter your current daily habits (sleep, diet, exercise, work, leisure, health), and the system will project your trajectory across different time horizons based on a statistical model:

- Age < 25 → 20-year projection
- Age 25–35 → 15-year projection
- Age 35–45 → 10-year projection
- Age 45–55 → 8-year projection
- Age > 55 → 5-year projection

### 2. Find Your Core Parameter
After analyzing your current state, the system identifies **1 core parameter** that has the greatest leverage on your life — not vague advice, but precision targeting.

### 3. Personalized Micro-Actions
Based on your real data and lifestyle patterns, it generates **3 micro-actions so easy they are almost impossible to fail**. Each action is:
- Extremely low barrier — completable in under 2 minutes
- Tied to an existing habit — no extra willpower needed
- Calibrated to your score gradient (lower scores = more foundational actions)

---

## 🎯 Key Features

- **Data-Driven**: Built on statistical models — no beautification, no toxic positivity
- **Age-Adaptive**: Projection time horizon dynamically adjusts with age
- **Honest Trajectories**: Good habits → upward; bad habits → downward
- **Fully Private**: All data stored locally in localStorage only
- **No Login Required**: Open and use immediately, zero friction

---

## 🛠 Tech Stack

- **Framework**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Typography**: Georgia (English/numbers) + Noto Serif SC (Chinese)
- **Charts**: Custom Canvas rendering
- **Storage**: localStorage (purely local, no backend)

## 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/sxxxxx11/life-modelling.git
cd life-modelling

# Install dependencies
npm install

# Start dev server
npm run dev
```
